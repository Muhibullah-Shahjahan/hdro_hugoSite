Open projects

We usually have a number of open research projects for MSc students who want to do an internship or graduation project. Please consult the list below or the TU/e Master Project Marketplace for a (more elaborate) set of available projects. More generally, if you have your own ideas for a brain-inspired AI Master project, feel free to contact prof. Bert de Vries.

  • MSc graduation project: Intelligent behavior through cooperating agents

    In this project you will study how two active inference agents can cooperate to solve a task.

  • MSc graduation project: Bayesian machine learning on a neuromorphic chip

    In this project you will simulate a Bayesian machine learning algorithm on a neuromorphic chip.

  • MSc graduation project: System identification with Bayesian machine learning

    In this project you will develop an agent to infer the dynamical parameters of an electro-mechanical positioning system.

  • MSc graduation project: an intelligent agent that learns to read text

    In this project you are challenged to develop a software demonstrator for an intelligent agent that learns how to read text.

  • MSc graduation project: Solving the Cocktail Party problem with Intelligent Autonomous Agents

    In this project, you are challenged to design an agent that learns to solve the cocktail party problem through on-the-spot interactions with a (human) listener.

  • MSc graduation project: Machine Learning for Human Motion Recognition

    In this project, you are challenged to develop novel machine learning technology for recognizing human motions.
